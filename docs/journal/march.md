# 3月2日
> 忙碌起来才会忘记痛苦吧  

3月15之前要急着上线公司项目，好在换来了一个能力还不错的后端，预期说是不错，倒不如说是也凑合，因为已经不能要求更多了，他做过7个项目，大抵也算运用成熟，只是他目前还是个比较自负的人。  
早上，前公司的财务帮我整理一些材料，以便能使我去申请大学生就业基层补贴，而之前一起创业的伙伴也推荐了个周末兼职授课的活儿，meanwhile，昨晚还接了个朋友的外包。  
今年还是希望忙碌一点，我已经不年轻了吧
